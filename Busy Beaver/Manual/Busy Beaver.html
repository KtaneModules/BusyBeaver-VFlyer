On the Subject of Busy Beaver

Turing's been busy.

  • On the module, there is a 2-character display and 10 buttons with ‘0’ on them.
  • Upon every non-ignored solved module, the 2-character display changes.
  • This display relates to a Turing operation to perform.
  • If the condition applies, make the digit 1 or move left respectively, otherwise, make it 0 or move right.
  • The first letter corresponds to the value to make the current digit, and the second on which way to move.
  • If the condition for the first letter returns true, make the current digit 1, or 0 otherwise.
  • If the condition for the second letter returns true, move left, otherwise, move right. The tape wraps around.
  • The value 'I' is the value of the currently selected tile.
  • The value 'P' is the currently selected tile's index (1 to 10).
  • The value 'TX' is the value of the X%10+1th tile.
  • The value 'S' is the stage number.

*All values on the tape start as 0.

Letter Condition Letter Condition
A I = 0 N I = 1
B P ≤ 5 O P ≥ 6
C TS = 1 P TS = 0
D P is Even Q P is Odd
E TS+P = 0 R TS+P = 1
F TP+5 = 1 S TP+5 = 0
G S is Even T S is Odd
H S+P is Odd U S+P is Even
I TS+P+I = 0 V TS+P+I = 1
J S-P+10 is Odd W S-P+10 is Even
K I+P is Even X I+P is Odd
L S-(P+I)+10 is Odd Y S-(P+I)+10 is Even
M TS+P+I+5 = 0 Z TS+P+I+5 = 1

Busy Beaver will ignore some modules.
